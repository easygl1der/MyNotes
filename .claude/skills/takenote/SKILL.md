# takenote skill

当你收到 `/takenote`、`记一下`、`帮我整理成笔记` 或类似指令时，触发本 skill。

## 触发条件
- 用户输入包含 "记一下"、"帮我整理"、"存到笔记"
- 用户发送附件路径并要求整理
- 用户输入 `/takenote`

## 处理流程
1. 解析用户输入内容
2. 按 CLAUDE.md 规则判断内容类型和存放位置
3. 搜索是否有相关已有笔记
4. 创建或更新笔记
5. 更新对应 _index.md
6. 告知用户结果

## 响应格式
每次处理完毕后，告知：
- 创建了什么文件
- 存放在哪个文件夹
- 添加了什么标签
- 是否更新了索引
